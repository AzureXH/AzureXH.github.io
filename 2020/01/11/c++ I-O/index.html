c++ I/O

摘抄于 geeksforgeeks

头文件

iostream

  • 用于标准的输入输出流
  • 包含了cin,cout,cerr函数

iomanip

  • I/O manipulators
  • 用于操作流的输出
  • 包含了setw, setprecision函数

fstream

  • 用于文件流 file stream
  • 用于处理从文件中作为input，读入数据
  • 也可用于写入文件作为ouput

cerr

  • 不经过缓存的标准错误流
  • 用于输出错误，同时也是一个ostream类的实例
  • 当需要立即展示错误信息时可以使用这个函数

clog

  • 经过缓存的错误流
  • 也是一个iostream类的实例，用于输出错误
  • 但与cerr不同的是，错误会首先插入到缓存中，并且存储在缓存中直到缓存被充满为止后放出。
