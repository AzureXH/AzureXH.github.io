c++ resize和reserve

resize

resize直接改变当前容器内含有元素的数量，但是不会改变原有位置的数字

resize(len)，如果len大于原先的size，会从size之后开始填充。

reserve

改变容器的最大容量，如果改变后的容量比改变前的要大，首先是会先分配一块新的内存，然后将原先的数据通过copy construct复制过去，再将之前的内存销毁。
